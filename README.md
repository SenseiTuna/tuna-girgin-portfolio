# Tuna Girgin Portfolio Website

Live Site: [tuna-girgin-portfolio.vercel.app](https://tuna-girgin-portfolio.vercel.app/)

## Overview

A modern bilingual portfolio website for software engineering internship, junior developer, R&D, computer vision, game development, and embedded systems opportunities.

The site presents selected projects, technical skills, education, work experience, contact links, and a downloadable CV in a responsive single-page experience.

## Features

- Turkish and English language support
- Responsive dark interface
- Interactive project detail modals
- Project-focused theme accents
- Downloadable CV
- GitHub and LinkedIn contact links
- SEO and Open Graph metadata

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Featured Projects

- Campus Vehicle System
- Olympus Voidbound
- TCDD Seat Finder Browser Extension
- Hardware Test & Automation Prototypes
